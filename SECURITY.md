# Security Policy

## Supported Versions

Security fixes are currently prioritized for:

- `0.9.x` beta line
- `main` branch

Older versions may not receive fixes.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Preferred process:

1. Open a private GitHub Security Advisory for this repository.
2. Include:
   - affected version/commit
   - reproduction details
   - impact assessment
   - any suggested remediation

If private advisories are not available, contact maintainers at `opensource@ella-project.dev`.

## What to Expect

- Initial triage acknowledgement target: within 5 business days
- We will validate impact, determine severity, and share remediation status
- We will coordinate disclosure timing with the reporter when possible
